# 🏙️ CivicSense | Empowering Citizens, Transforming Cities

CivicSense is a high-performance, real-time community reporting platform designed to bridge the gap between citizens and local authorities. 

![CivicSense Hero](https://github.com/Arrow2211/Civic-Sense/raw/main/hero.png) *(Placeholder: You can replace this with your project screenshot)*

## ✨ Key Features

*   **Premium Glassmorphism UI**: Built with Tailwind CSS and custom animations for a state-of-the-Art aesthetic.
*   **Real-time Intelligence**: Instant synchronization of community reports and authority resolutions via Supabase Realtime.
*   **Multi-Role Dashboards**:
    *   **Citizens**: Report issues with photo/video proof, track personal history, and see the impact.
    *   **Authorities**: Manage a real-time queue, resolve tasks with visual evidence, and monitor city-level statistics.
*   **Secure Authentication**: Dual-portal sign-in for Citizens and Officers with robust session management.

## 🚀 Deployment (Vercel)

This project is optimized for direct hosting on **Vercel**. To deploy:

1.  Connect your GitHub repository (**Arrow2211/Civic-Sense**) to Vercel.
2.  Ensure the "Build Command" is left empty (it's a static site!).
3.  Set the "Output Directory" to the root `./`.
4.  Launch!

## 🛠️ Stack

*   **Language**: Vanilla JavaScript (ES6+ Modules)
*   **Styling**: HTML5, Vanilla CSS, Tailwind Play CDN
*   **Database/Auth**: [Supabase](https://supabase.com)
*   **Icons**: Font Awesome 6.4

## 📂 Project Structure

*   `index.html`: Entry point and SPA views.
*   `style.css`: Custom animations and layout components.
*   `app.js`: Core business logic, Auth, and Supabase integration.
*   `setup.sql`: Database schema and realtime policies.
*   `vercel.json`: SPA routing configuration for Vercel.
